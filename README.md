# Tomato-Reset-CSS
Tomato Reset CSS UI

### 依赖工具包
* [Node](https://nodejs.org)
* [NPM](https://www.npmjs.com)
* [Grunt](http://gruntjs.com)

### 其它依赖
* Less

### Grunt 自动化
* load-grunt-tasks
* time-grunt
* grunt-browser-sync
* grunt-contrib-clean
* grunt-contrib-less
* grunt-autoprefixer

### 使用方法

```
// 引入 Tomato CSS 文件
<link rel="stylesheet" type="text/css" href="dist/css/tomato.min.css">
// 你的 CSS 文件
<link rel="stylesheet" type="text/css" href="yourCss.css">
```
